Accidents de la route

Dans un premier temps, il fallait explorer la base de données pour voir comment elle était construite et agencée pour que par la suite le nettoyage et l’exploitation soient plus facile. Pour la partie nettoyage, il fallait nettoyer plusieurs tables pour enlever les donner inutiles mais également pour réunir les données de 2006 à 2021 car dans la base de données proposée, les données n’étaient pas réunies. Donc pour cela, nous avons créé des vues pour stocker ces nouvelles données réunissant toutes les années, mais dans les tables de 2006 à 2018 le type de certains attributs n’était pas le même que dans les tables de 2019 à 2021 et nous ne pouvions donc pas uniquement les joindre. Il nous a donc fallu convertir tous les types en utilisant des fonctions de conversion. De plus, les dates dans certaines tables étaient en format jj/mm/aa et dans d’autres le format était jj/mm/aaaa ce qui nous a donc obligé à tout mettre dans le même format afin de pouvoir les réunir correctement.

Une fois l’étape de nettoyage faite, nous avons fait de l’extraction de données selon notre cas. Notre cas était d’analyser les données de la région Nouvelle-Aquitaine sur les personnes de moins de 18 ans. Pour cela on a créé des tables temporaires et des vues afin de stocker ces données. Nous avions une première table dans laquelle on pouvait retrouver toutes les caractéristiques des victimes mineures d’accidents de la route sur la région Nouvelle-Aquitaine, telles que le numéro d’accident, le numéro de véhicule, le lieu, le jour, si l’accident a eu lieu le matin ou l’après-midi etc. Une deuxième table sur les véhicules afin de savoir quels étaient les types de véhicules impliqués dans l’accident comme la voiture, le vélo, le tram etc. Puis une dernière table sur le lieu afin de connaître les informations des victimes pour chaque département de la région. Après cette étape d’extraction, nous avons fait quelques tables qui étaient le nombre d’hommes victimes,le nombre de femme, le nombre de personnes tuées, le nombre de personnes blessées, le nombre de personnes hospitalisées qui ont permis à la fin de créer une table bilan avec les seulement les informations nécessaires. Une fois toutes les tables créées, nous avons exporter les données dans un document csv à savoir excel pour l'étape de statistiques.

Enfin, nous avons produit des graphes ainsi que leurs commentaires en anglais car l’objectif final était de produire un document destiné à avertir et prévenir les usagers de la route sur les accidents de la route.
